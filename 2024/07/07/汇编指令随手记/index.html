汇编指令随手记

JnamerZ 2024-07-07 20:45:10
Categories： ctf > 编程语言 Tags： ctf 编程语言 汇编语言

开头

本篇将记录一些关于汇编指令的一些东西，会动态更新

寄存器

不同架构下寄存器名称可能会不一样，这里用一些比较通用的符号来表示相关的寄存器

常用指令

mov

将内存或寄存器中的数值进行拷贝

jxx

当满足某一条件时，修改ip的跳转到某个地址，其中jmp是无条件跳转

常用于跳转到shellcode，如jmp rax

push/pop

将操作数压栈和弹出的指令，push ax等价于下面两条指令：

  1. sub sp, X 其中X=CPU位宽÷8
  2. mov [sp], ax

pop ax等价于下面两条指令：

  1. mov ax, [sp]
  2. add sp, X 其中X=CPU位宽÷8

call

call ptr指令做了两件事：

  1. 将ptr压入栈中
  2. 跳转到ptr

关于call的花指令可以根据这一点进行分析，也常用于调用shellcode

leave

等价于下列两条指令

  1. mov sp, bp
  2. pop bp

ret

等价于弹出栈顶的值，并将其复制到到ip中

add

将两个操作数相加，并将结果存入其中一个。如add rax, 1就是将rax中的值与1相加，结果存入rax中

sub

就是add的减法版本

shl/shr/sal/sar/rol/ror/rcl/rcr

对操作数进行位移操作，用法都是xxx dest, count，其中dest是目标操作数，count是移动的位数

指令 操作
shl 逻辑左移
shr 逻辑右移
sal 算术左移
sar 算术右移
rol 循环左移
ror 循环右移
rcl 带进位的循环左移
rcr 带进位的循环右移

and/or/xor

将两个操作数进行位运算，并将结果存入其中一个。

如and rax, rbx就是将rax中的值与rbx中的值取与，结果存入rax中

如xor rsi, 0x101就是将rax中的值与0x101异或，结果存入rax中

xor常用于清空寄存器，如xor rax, rax

not/neg

not是对操作数取非并存入其中

neg是取操作数补码并存入其中

nop

什么也不做，只会让ip继续向后移动

cmc

将标志位CF取反

clc

将标志位CF置0

stc

将标志位CF置1

cli

仅内核态可用

将标志位IF置0，关闭中断

sti

仅内核态可用

将标志位IF置1，打开中断

cld

将标志位DF置0

std

将标志位DF置1

单字节指令

用于构造一些复杂的shellcode，这里只列出来不做详细解释

  • nop 144 0x90
  • cmc 245 0xf5
  • clc 248 0xf8
  • stc 249 0xf9
  • cli 250 0xfa (kernel mode only)
  • sti 251 0xfb (kernel mode only)
  • cld 252 0xfc
  • std 253 0xfd
  • pop ax/bx/cx/dx/di/si/sp/bp
  • ret 0xc3
  • leave 0xc9
